# Source Vocabulary Data

This folder keeps source vocabulary files used for glossary decisions.

## CEFR A1

- Source: Oxford Learner's Dictionaries, Oxford 3000 and 5000
- Page: https://www.oxfordlearnersdictionaries.com/wordlists/oxford3000-5000
- PDF: https://www.oxfordlearnersdictionaries.com/external/pdf/wordlists/oxford-3000-5000/The_Oxford_3000_by_CEFR_level.pdf
- Extracted files:
  - `cefr/oxford3000_a1_words.txt`
  - `cefr/oxford3000_a1_words.csv`

CEFR A1 words are treated as basic words for this project and normally do not receive footnotes.
